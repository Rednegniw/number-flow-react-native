# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.3.x | Yes |
| < 0.3 | No |

## Reporting a Vulnerability

If you discover a security issue, please report it responsibly. Do not open a public issue.

Contact [@Rednegniw on GitHub](https://github.com/Rednegniw) with:

- A description of the vulnerability
- Steps to reproduce
- Affected versions
- Potential impact

## Scope

Number Flow React Native is a client-side UI animation library. It does not handle authentication, network requests, or sensitive data storage.

Relevant security concerns include:

- Dependency vulnerabilities (supply chain)
- Injection via rendered content
- Unsafe dynamic code execution

## Disclosure Policy

We follow coordinated disclosure: confirm the issue, prepare and release a fix, then publicly disclose with credit to the reporter.
