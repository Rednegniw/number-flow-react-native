title: Currency Formatting
description: Animated currency display with locale-aware formatting

---

`NumberFlow` accepts `Intl.NumberFormatOptions` via the `format` prop. Combined with `locales`, you get locale-specific currency symbols, decimal separators, and grouping, all animating independently.

```LiveExample
import { useState } from 'react';
import { View } from 'react-native';
import { NumberFlow } from 'number-flow-react-native';
import { ExampleLayout, Button } from './ExampleUI';

const currencies = [
  { code: 'USD', locale: 'en-US', label: '$ USD' },
  { code: 'EUR', locale: 'de-DE', label: '€ EUR' },
  { code: 'JPY', locale: 'ja-JP', label: '¥ JPY' },
];

function randomize() {
  const amount = Math.floor(Math.random() * 9999) + 1;
  const index = Math.floor(Math.random() * currencies.length);
  return { amount, index };
}

export default function CurrencyFormatting() {
  const [{ amount, index }, setState] = useState(() => randomize());

  const currency = currencies[index];

  return (
    <ExampleLayout>
      <NumberFlow
        value={amount}
        format={{ style: 'currency', currency: currency.code }}
        locales={currency.locale}
        style={{ fontSize: 40, color: '#fff' }}
      />

      <Button title="Randomize" onPress={() => setState(randomize())} />
    </ExampleLayout>
  );
}
```

### Key points

- The `format` prop is passed directly to `Intl.NumberFormat`. Any option that works with `Intl.NumberFormat` works here.
- When switching currencies, symbols like `$`, `€`, and `¥` animate in and out with opacity and vertical slide transitions.
- The `locales` prop controls grouping separators (`1,234.56` in `en-US` vs `1.234,56` in `de-DE`) and currency symbol placement.
