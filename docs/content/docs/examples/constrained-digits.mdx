---
title: Constrained Digits
description: Per-position digit constraints for clocks, timers, and custom ranges
full: true
---

The `digits` prop limits individual digit positions to custom ranges. Instead of the default 0–9 wheel, you can constrain a position to any maximum between 1–9. Here, every digit is constrained to `{ max: 1 }`, turning each position into a binary flip (0 or 1).

```LiveExample
import { useState } from 'react';
import { View, Text } from 'react-native';
import { NumberFlow } from 'number-flow-react-native';
import { ExampleLayout, Button } from './ExampleUI';

const BITS = 8;

const binaryDigits = Object.fromEntries(
  Array.from({ length: BITS }, (_, i) => [i, { max: 1 }])
);

function toBinaryNumber(n) {
  return parseInt(n.toString(2));
}

function randomByte() {
  return Math.floor(Math.random() * 256);
}

export default function BinaryCounter() {
  const [decimal, setDecimal] = useState(42);

  const binaryValue = toBinaryNumber(decimal);

  return (
    <ExampleLayout>
      <View style={{ alignItems: 'center', gap: 4 }}>
        <Text style={{ fontSize: 14, color: 'rgba(255,255,255,0.5)' }}>Binary</Text>
        <NumberFlow
          value={binaryValue}
          digits={binaryDigits}
          format={{ minimumIntegerDigits: BITS, useGrouping: false }}
          style={{ fontSize: 36, color: '#fff', fontFamily: 'JetBrains Mono, monospace' }}
        />
        <View style={{ flexDirection: 'row', alignItems: 'center' }}>
          <Text style={{ fontSize: 14, color: 'rgba(255,255,255,0.5)' }}>decimal: </Text>
          <NumberFlow
            value={decimal}
            style={{ fontSize: 14, color: 'rgba(255,255,255,0.5)' }}
          />
        </View>
      </View>

      <View style={{ flexDirection: 'row', gap: 8 }}>
        <Button title="-1" onPress={() => setDecimal(v => Math.max(0, v - 1))} />
        <Button title="+1" onPress={() => setDecimal(v => Math.min(255, v + 1))} />
        <Button title="Random" onPress={() => setDecimal(randomByte())} />
      </View>
    </ExampleLayout>
  );
}
```

### Notes

- `{ max: 1 }` on every position creates a 2-element wheel (0, 1) instead of the default 10-element wheel (0–9). Each bit flips with a short, snappy roll.
- The decimal value is converted to its binary representation via `parseInt(n.toString(2))`, then `minimumIntegerDigits: 8` pads to a full byte.

<Callout type="info">
`TimeFlow` automatically applies digit constraints internally (minutes tens: 0–5, hours tens: 0–2). The `digits` prop is for `NumberFlow` when you need custom constraints.
</Callout>
